用 VERCEL (ZEIT) 搭建你的静态网站

May 18 2020

使用 vercel 搭建静态网站，相比于 github，其在大陆的速度更为理想，而且有 CDN 加速。那么今天我们就来搭建一个静态网站。

首先使用 hexo, hugo 或者其它工具生成网页，然后在 GitHub 新建一个项目，最后将生成工具生成的文件 push 到 GitHub 上面去就可以了。

然后我们在 VERCEL 使用 GitHub 注册一个账号之后，开始下面的操作。

在主界面我们点击 import project

然后选择 From Git Repository, 点击 Continue.

我们选择从 GitHub 导入项目

最后选择你刚才创建的项目就可以了

最后按着指示，一直 continue 就可以了。

接下来你就可以在 VERCEL 看到你的项目。
