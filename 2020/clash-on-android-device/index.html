在安卓手机上使用 Clash 科学上网

May 19 2020 科学上网

本文最近更新 20200524

在前面我们讲过如何在 Windows 电脑上使用 Clash 进行科学上网，现在我们来学习一下如何在安卓手机上使用 Clash。过程几乎和 Windows 上的没有区别。

你还可以在 Play Store 上进行下载，下载地址：https://play.google.com/store/apps/details?id=com.github.kr328.clash

你还可以直接在 GitHub 上下载：https://github.com/Kr328/ClashForAndroid/releases

其实在安卓手机上使用 Clash 的操作和电脑上差不多，我们还是要从机场获取 Clash 的订阅链接。

然后进入刚才下载的软件，点击 Profiles

进去之后点击 + New Profile ,然后点击 URL(Import from URL)

然后复制链接即可

点击 Tap to start 开启 Clash, 进入 Proxy 选择节点

推荐使用 Just My Socks 进行科学上网。
