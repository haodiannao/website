用 VERCEL (ZEIT) 搭建你的静态网站

May 18 2020 服务器

使用 vercel 搭建静态网站，相比于 github，其在大陆的速度更为理想，而且有 CDN 加速。那么今天我们就来搭建一个静态网站。

首先使用 hexo, hugo 或者其它工具生成网页，然后在 GitHub 新建一个项目，最后将生成工具生成的文件 push 到 GitHub 上面去就可以了。

我们在 VERCEL 使用 GitHub 注册一个账号之后，开始下面的操作。

在主界面我们点击 import project

选择 From Git Repository, 点击 Continue.

我们选择从 GitHub 导入项目

最后选择你刚才创建的项目就可以了

最后按着指示，一直点击 continue

接下来你就可以在 VERCEL 看到你的项目，点击 Visit 我们就可以访问我们的网站

如果想配置自定义域名就可以回到上一个图片的界面，然后点击项目名称，然后点击 Settings > Domains

来到这一界面，添加你注册的域名

更改 Nameserver 如下面的即可

a.zeit-world.com
b.zeit-world.net
c.zeit-world.org
e.zeit-world.co.uk

Vercel 会保持你以前的 DNS 配置，不过需要等待一段时间。当 Vercel 检测你的 Nameserver 更改完成之后，你就可以用域名访问你的网站了。
